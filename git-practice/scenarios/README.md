# Git Scenarios - Practice Exercises

This directory contains practical scenarios you'll encounter in real-world collaborative Git workflows.

## How to Use

1. Read each scenario description
2. Try to solve it yourself first
3. Check the solution if needed
4. Practice multiple times until comfortable

## Tips

- Create a backup branch before attempting risky operations
- Use `git reflog` as a safety net
- Test solutions in this practice repo first
- Don't be afraid to make mistakes - that's how you learn!
